# Console-Based-PK-GK-Quiz-CPP
A terminal-based Pakistan General Knowledge quiz in C++ with 50 questions, random selection, and score tracking.
A terminal-based General Knowledge quiz about Pakistan, written in C++.
The user chooses how many questions they want (out of 50), random questions
appear, and at the end the score and percentage are displayed.

## Features
- 50 General Knowledge questions about Pakistan
- User chooses how many questions to attempt (1–50)
- Questions are randomly selected every session
- Score and percentage shown at the end
- No dependencies — pure C++

## How to Compile & Run
g++ -o QuizOnConsole QuizOnConsole.cpp
./QuizOnConsole

## How to Play
1. Run the program
2. Enter how many questions you want
3. Answer each question
4. See your final score and percentage

## Language
C++
